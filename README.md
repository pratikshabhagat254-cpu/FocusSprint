# 🚀 FocusSprint

### A modern productivity dashboard for managing study tasks, deadlines, and productivity

FocusSprint is a web-based productivity dashboard designed to help students organize academic tasks, manage deadlines, and monitor progress through a clean and intuitive interface.

Built using **HTML, CSS, and JavaScript**, the project focuses on practical productivity features while applying core frontend development concepts such as dynamic rendering, local storage persistence, responsive design, and user-centered dashboard development.

🌐 **Live Demo:** https://focus-sprint-ten.vercel.app/

🔗 **Repository:** https://github.com/pratikshabhagat254-cpu/FocusSprint

---

## 📌 Overview

Managing assignments, study goals, and deadlines across multiple subjects can quickly become challenging.

FocusSprint provides a centralized workspace where students can:

* Organize study tasks
* Track upcoming deadlines
* Prioritize academic work
* Monitor productivity progress
* Maintain consistency in their study routine

Inspired by modern productivity dashboards, the application aims to deliver a focused, responsive, and user-friendly experience for students.

---

## ✨ Features

### 📋 Task Management

* Create, edit, and delete tasks
* Mark tasks as completed
* Assign subject/category tags
* Set task priorities (High, Medium, Low)
* Organize academic workload efficiently

### ⏰ Deadline Tracking

* Add due dates to tasks
* Monitor upcoming deadlines
* Identify overdue tasks
* View task countdown status

### 📊 Productivity Dashboard

* Total task overview
* Completed task count
* Pending task count
* Progress tracking and completion insights

### 🔍 Search & Filtering

* Search tasks instantly
* Filter completed tasks
* Filter pending tasks
* Filter high-priority tasks
* Sort tasks by deadline

### 💾 Data Persistence

* Local Storage integration
* Tasks remain available after page refresh
* No account setup required

### 📱 Responsive Design

* Optimized for desktop and laptop screens
* Responsive layout for tablets and mobile devices
